# orthopedic-platform-BoniCare-
Graduation Project

Graduation Project (Backend)

This repository contains the backend for the BoniCare orthopedic platform (Node.js + Express + MongoDB).

Quick start

1. Copy `.env.example` to `.env` and set your environment variables.
2. Install dependencies:

```powershell
npm install
```

3. Start the server:

```powershell
npm start
```

Notes

- The app entrypoint is `server.js`.
- Project uses ES modules (`type: "module"` in `package.json`).

GitHub

- Branching strategy: create feature branches from `main` or `backend/main`, then open PRs to `main`.
- A basic GitHub Actions workflow is included at `.github/workflows/nodejs.yml`.
